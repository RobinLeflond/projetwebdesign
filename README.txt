FORTNITE LOOTS

Ce site a pour objectif de présenter une liste des armes pouvant êtres obtenues en tant que "Loot" (Butin) dans le jeu Fortine.

Description
=================
Le site contient une entête où se trouve le titre du site, deux images et une barre de recherche puis un menu constitué de deux onglets.
Le premier onglet permet d'afficher une liste des armes Fortnite avec leurs spécificités (dégats, temps de chargement, etc.).
Cette liste peut être triée par chacun des critères définissant les armes (nom, rareté, dégâts, dégats lors d'un head shot, tirs par seconde, dégâts par seconde, chargeur et charges).
Un bouton permet d'ajouter chaque arme à une liste des favoris accessible dans le second onglet.
Dans ce second onglet, un bouton permet d'afficher ou de cacher les images des armes favorites.
De plus un bouton "supprimer" permet de supprimer une arme de cette liste.
Enfin il est possible de rechercher dans les deux listes une arme particulier en rentrant son nom dans la barre de recherche.

Datasource
=================
Les données utilisées par le site ont été acquises à partir de différents tableaux présents sur le site : https://www.millenium.org/guide/278927.html puis modifiées au format au JSON.

Fonctionnalités
=================
Cette application utilise les directives du core d’AngularJS ng-model, ng-if, ng-show, ng-repeat, ng-class et ng-click.
L'application utilise des fonctionnalités graphique de Bootstrap.
Un filtre personnalisé est utilisé pour renommer la rareté des armes.

Style et Design
=================
Les couleurs du site ont été inspirées de la palette de couleur utilisée par le jeu Fortnite.
De même l'icône de notre application est la même que celle du site officiel du jeu.
Les couleurs des lignes des tableaux sont définies en fonction de la rareté de l'arme.
Différentes animations ont été configurées pour les onglets et boutons :
-Transitions progressives
-Modification des couleurs lors du changement d'onglet
-Animation des boutons lors du clique

L'affichage des éléments diffère en fonction de la taille de l'écran, en effet les éléments suivants apparaissent différemment :
- la taille de la barre de recherche
- la taille de l'entête
- l'affichage ou non des images de l'en-tête
- la taille occupée latéralement par le contenu

Version
=================
Il est recommandé pour le bon fonctionnement du site d'utiliser le navigateur de recherche Mozilla Firefox.
